# tailwind-init

> Install Tailwind CSS to a vanila HTML. Included: postcss-nested, live-server, css-nano and purge-css

## Usage

```bash
tailwind-init dir-name
cd dir-name

// start developing
npm run dev
yarn dev

// purge css
npm run build
yarn build

// purge css and minimize
npm run prod
yarn prod
```
